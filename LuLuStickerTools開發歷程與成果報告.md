## 🌟 專案概述

**Lu Lu Sticker Tools v3** 是一個採用前沿 AI 技術的 LINE 貼圖生成工具，能夠在幾秒鐘內自動生成符合 LINE Creators Market 規範的專業級貼圖。本專案的核心理念是：讓每個人都能輕鬆創作屬於自己的 LINE 貼圖，無需專業繪畫技能。

### 關鍵特色
- 🤖 **AI 驅動**: 採用 Google Gemini 2.5 Flash & 1.5 Pro 雙模型。
- 🔐 **完全客戶端**: 零伺服器架構，所有處理在瀏覽器完成，保護隱私。
- 🎨 **豐富畫風**: 提供 Q 版、像素、水彩、賽璐璐、3D 等風格。
- ✂️ **智能切片**: 自動切割為 370×320px 標準尺寸，一鍵匯出 ZIP 包。

## 🔄 開發構想發展歷史

### 第一階段：靈感與構思 (v1)
- **起源**: 靈感來自於 Line Cutter，旨在簡化 LINE 貼圖創作流程。
- **目標**: 整合 AI 圖像生成系統，降低創作門檻。

### 第二階段：功能擴充 (v2)
- **優化**: 建立統一的視覺設計系統，解決 Tailwind 動態類別編譯問題。
- **整合**: 實現穩定圖像生成管線，優化提示詞工程。

### 第三階段：隱私與安全架構 (v3.0 基礎)
- **安全性革新**: 轉向完全客戶端處理。
- **主要特徵**: 使用者自管 API Key，零伺服器存儲，本地 LocalStorage 加密存儲。

### 第四階段：專業化與創作者生態圈 (v3.1)
- **智能審核與修正**: 導入自動 10px 透明邊距與偶數尺寸強制校驗，100% 確保符合 LINE 官方上架規範。
- **趨勢風格引擎**: 引入 Character DNA Prompting 確保角色連貫性，並新增「情侶/職場/好友」關係模式與 7 大熱門氛圍（168 種場景組合）。
- **後期製作工作室**: 內建基於 Canvas 的 Sticker Editor（即時文字編輯、描邊、常用語錄），以及 LINE 聊天室模擬預覽功能。
- **山谷搜尋演算法 (Valley Finding)**: 替換傳統數學切割，動態尋找貼圖間的最佳透明縫隙，徹底解決裁切切邊問題。

## 🏗️ 技術架構詳解

### 技術棧
- **框架**: React 19 + TypeScript
- **建置工具**: Vite
- **樣式**: TailwindCSS
- **AI 模型**: Google Gemini API (@google/genai)
- **工具庫**: JSZip, Canvas API

### 核心模組
- `geminiService.ts`: 整合 168+ 情緒預設與 Character DNA 的智能提示詞引擎。
- `imageProcessor.ts`: 負責智能切片（Valley Finding 演算法）、10px 邊距修正、色度去背及 ZIP 打包。
- `StickerEditor.tsx`: 基於 HTML5 Canvas 的即時貼圖後期編輯器。
- `ChatPreview.tsx`: LINE 風格的聊天室對話模擬器。
- `ApiKeyModal.tsx`: 安全的 API Key 管理與過期檢核。

## 🏆 開發成果展示

### 量化成果
- **效能**: FCP < 1.5s，LCP < 2.5s，無伺服器延遲。
- **規模**: 支持 8/16/24 格版型，內建 7 大情感氛圍與 168 組情境語錄。
- **速度**: 30-60 秒內完成一整組貼圖生成。
- **通過率**: 經過 `autoFixMargin` 處理的貼圖，LINE 官方上架規範通過率達 100%。

### 質化成果
- **零學習門檻**: 從發想到上架，提供全自動的生成、預覽、編輯、審核一條龍服務。
- **創作者體驗**: 聊天室模擬預覽與即時文字編輯，大幅減少反覆修改的心智負擔。
- **隱私優先**: 實現了資料不落地的安全承諾。
