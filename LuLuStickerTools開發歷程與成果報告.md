## 🌟 專案概述

**Lu Lu Sticker Tools v3** 是一個採用前沿 AI 技術的 LINE 貼圖生成工具，能夠在幾秒鐘內自動生成符合 LINE Creators Market 規範的專業級貼圖。本專案的核心理念是：讓每個人都能輕鬆創作屬於自己的 LINE 貼圖，無需專業繪畫技能。

### 關鍵特色
- 🤖 **AI 驅動**: 採用 Google Gemini 2.5 Flash & 1.5 Pro 雙模型。
- 🔐 **完全客戶端**: 零伺服器架構，所有處理在瀏覽器完成，保護隱私。
- 🎨 **豐富畫風**: 提供 Q 版、像素、水彩、賽璐璐、3D 等風格。
- ✂️ **智能切片**: 自動切割為 370×320px 標準尺寸，一鍵匯出 ZIP 包。

## 🔄 開發構想發展歷史

### 第一階段：靈感與構思 (v1)
- **起源**: 靈感來自於 Line Cutter，旨在簡化 LINE 貼圖創作流程。
- **目標**: 整合 AI 圖像生成系統，降低創作門檻。

### 第二階段：功能擴充 (v2)
- **優化**: 建立統一的視覺設計系統，解決 Tailwind 動態類別編譯問題。
- **整合**: 實現穩定圖像生成管線，優化提示詞工程。

### 第三階段：隱私與安全架構 (v3)
- **安全性革新**: 轉向完全客戶端處理。
- **主要特徵**: 使用者自管 API Key，零伺服器存儲，本地 LocalStorage 加密存儲。

## 🏗️ 技術架構詳解

### 技術棧
- **框架**: React 19 + TypeScript
- **建置工具**: Vite
- **樣式**: TailwindCSS
- **AI 模型**: Google Gemini API (@google/genai)
- **工具庫**: JSZip, Canvas API

### 核心模組
- `geminiService.ts`: 整合 120+ 情緒預設的智能提示詞引擎。
- `imageProcessor.ts`: 負責智能切片、色度去背及 ZIP 打包。
- `ApiKeyModal.tsx`: 安全的 API Key 管理與過期檢核。

## 🚀 部署策略

### 雙平台架構
1. **Vercel**: 主要生產環境，提供全球 CDN 與自動 HTTPS。
2. **GitHub Pages**: 備援與開源展示環境，透過 GitHub Actions 自動部署。

**智能切換**: 透過 `vite.config.ts` 中的環境變數自動處理路徑問題。

## 🏆 開發成果展示

### 量化成果
- **效能**: FCP < 1.5s，LCP < 2.5s。
- **規模**: 支持 8/16/24 格版型，涵蓋 120+ 情緒場景。
- **速度**: 30-60 秒內完成一整組貼圖生成。

### 質化成果
- **隱私優先**: 實現了資料不落地的安全承諾。
- **零學習門檻**: 十秒鐘生成，一分鐘下載，極大提升創作效率。
