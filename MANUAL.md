# Lu Lu Sticker Tools v3 使用手冊

本手冊引導您如何從零開始使用 Lu Lu Sticker Tools 生成並打包一組符合 LINE 官方規範的貼圖。

## 📥 快速啟動
1. 訪問 [線上版本](https://lu-lu-sticker-tools-v3.vercel.app) 或本地運行 `npm run dev`。
2. **輸入 API Key**：前往 [Google AI Studio](https://aistudio.google.com/app/apikey) 獲取免費的 Gemini API Key。
3. **設定有效期限**：建議設定 30 天，系統會將 Key 安全地儲存在您的瀏覽器本地（LocalStorage）。

## 🎨 製作流程

### 第一步：風格定義
*   **選擇版型**：
    *   **8 格**：適合快速製作一組常用表情。
    *   **16 格**：主流選擇，適合完整展現角色情緒。
    *   **24 格**：專業級別，提供最豐富的選擇。
*   **選擇畫風**：
    *   **🎨 經典 Q 版**：2D平面粗線條，通過率與穩定度最高。
    *   **🥴 潦草醜萌風 (2026 最夯趨勢)**：隨性塗鴉、歪斜線條，傳遞年輕世代精神鬆弛感與隨意態度。
    *   **🥺 軟萌療癒風**：吉伊卡哇式溫暖，粉嫩配色與委屈大眼。
    *   **💿 Y2K 千禧潮流**：高對比酸性色調、8-bit 像素或復古美式。
*   **選擇氛圍 (Vibe Presets)**：系統內建多套時事流行情境情節，會自動生成對應的台詞與角色動作：
    *   **🔥 脆式時事熱梗 (Threads Memes)**：觸爛、留友看、接住你、建議手臂加強、真冰涼。
    *   **🫠 精神登出社畜 (Office Mentally Checked Out)**：表面禮貌內心發瘋、精神登出、暈碳、已閱讀。
    *   **🥺 委屈討拍牽牽 (Needy & Clingy)**：牽牽、求關注、抱緊處理、撒嬌討拍拍。
    *   **🤪 荒謬反差幽默 (Absurd Humor)**：確？、供三小、美麗精神狀態、笑到往生。

### 第二步：角色描述與參考圖
*   **文字描述**：請儘量具體。
    *   *❌ 錯誤範例*：「一隻棕色的小狗」
    *   *✅ 正確範例*：「一隻穿著黃色雨衣的棕色柴犬，臉頰圓圓的，尾巴捲曲，風格簡潔可愛。」
*   **參考圖（選填）**：如果您已有角色草稿，上傳它能顯著提升 AI 生成的一致性。

### 第三步：生成與精修
1. 點擊 **「開始生成」**。生成過程約需 30-60 秒（取決於模型負載）。
2. **單張重繪**：若某個格子生成效果不佳，點擊該格子左上角的「循環圖示」即可針對該格重新生成。
3. **手動替換**：您也可以點擊「上傳圖示」，將自己繪製的圖片替換進去。

### 第四步：導出與打包
1. **去背處理**：勾選「移除綠色背景」以獲得透明底貼圖。
2. **檢查預覽**：切換下方的背景顏色（白、黑、紅、透），檢查邊緣是否有雜訊。
3. **下載 ZIP**：點擊「下載完整 ZIP 包」。
    *   該包內已包含 `main.png` (240x240)、`tab.png` (96x74) 以及正確編號的貼圖 (370x320)。

## 🛠 進階技巧
*   **模型選擇**：
    *   快速測試時使用 **Gemini 2.5 Flash**。
    *   製作最終成品時務必切換至 **Gemini 1.5 Pro** 以獲得細節最精緻的效果。
*   **安全邊距**：系統已強制加入 10px 的安全邊距，請勿再手動裁切邊緣，以免違反 LINE 審核規範。

## ❓ 常見問題
*   **為什麼生成失敗？**
    *   請檢查您的 API Key 是否已達到免費配額上限（每分鐘 2 次生成）。
    *   確認描述詞是否包含敏感內容。
*   **圖片模糊？**
    *   請確保選擇的是 1.5 Pro 模型，並檢查網絡連接是否穩定。
